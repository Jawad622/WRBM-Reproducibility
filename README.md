# WRBM-Reproducibility

## Overview
This repository reproduces results for WRBM model on similarity (MC, RG65, WS203, SimLex) and relatedness bechmarks (353ALL, MTurk287, MTurk771, MEN3000).

## Data
This code uses following files:
1. Relatedness_Benchmarks
2. Similarity_Benchmarks
3. AlldsTokenDF

## Installation
pip install -r requirements.txt

## How to Run
The original code was run on python 3.10. After installing requirements.txt, execute the code to compute SS and SR between WordNet concepts.

## Outputs
Results will be saved in:
WRBM_Results/
